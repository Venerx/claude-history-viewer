# Claude History Viewer

Public starter repository for a local Claude history browser.

What it is:

- code-only starter for the app
- no private Claude export data included
- ready for your own `source/conversations.json`

Quick start:

1. Put your export in `source/conversations.json`
2. Run `python3 app.py`
3. Open `http://127.0.0.1:5174`

Included features:

- full-text search
- conversation views
- pinned threads
- content tabs for conversations and artifacts
- transient quick views for media, attachments, memories, and projects
- artifact and media jump-back flows
- attachment repair/upload workflow
